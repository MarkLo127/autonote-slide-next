## Autonote&Slide


### 1. 克隆專案

```bash
git clone https://github.com/MarkLo127/autonote-slide-next.git
```

### 2.進入專案目錄
```bash
cd autonote-slide-next
```

### 3.安裝與啟動前端
```bash
pnpm -C frontend i
pnpm -C frontend dev
```

### 4. 安裝與啟動後端
```bash
pip install -r backend/requirements.txt
python -m backend
```
